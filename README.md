## Project snapshot

This repository is structured as an end-to-end Data Science case study:

- **Business/problem framing**: identify which passenger attributes were most associated with survival
- **Exploratory Data Analysis**: uncover patterns across sex, class, age, fare, and embarkation port
- **Feature engineering**: create stronger model inputs such as family size, title, age groups, and derived fare features
- **Modeling**: train and compare baseline classification models
- **Communication**: save charts in `reports/figures/` and document the analysis clearly for recruiters

Supporting questions:

- Did women survive at higher rates than men?

## Analysis workflow

### 1. Data understanding
Inspect the Titanic dataset, variable types, target distribution, and missing values.

### 2. Cleaning and feature engineering
Standardize column names, handle nulls, create family-size and age-based features, and improve the modeling dataset.

### 3. Exploratory Data Analysis
Study survival patterns by sex, class, age, fare, and embarkation point.

### 4. Modeling
Train baseline classifiers such as Logistic Regression and Random Forest.

### 5. Evaluation
Measure performance using accuracy, precision, recall, f1-score, and a confusion matrix.

### 6. Reproducibility
Save charts into `reports/figures/` and generate a plot manifest for easy tracking.

## How to run the project

### 1. Clone the repository
```bash
git clone https://github.com/mayconserzedelo/titanic-data-science.git
cd titanic-data-science
```

### 2. Create a virtual environment
```bash
python -m venv venv
```

### 3. Activate the environment

**Windows**
```bash
venv\\Scripts\\activate
```

**macOS / Linux**
```bash
source venv/bin/activate
```

### 4. Install dependencies
```bash
pip install -r requirements.txt
```

### 5. Add the data file
Place the Titanic training dataset at:

```bash
data/raw/train.csv
```

### 6. Open the notebook
```bash
jupyter notebook
```
